Privacy Policy

Last updated: October 2025

Overview

PinGPT is a Chrome extension that allows users to pin, name, and search messages within ChatGPT. This extension is designed with privacy in mind — all data is stored locally in your browser and never transmitted to any external server.

Information We Collect

PinGPT does not collect, transmit, or store any personal information, chat content, or usage data on remote servers. All pinned message data, names, and tags are stored locally in your browser using IndexedDB and remain on your device.

How Your Data Is Used

  • To display your saved pins inside the extension popup.
  • To allow you to search, edit, export, or delete pins.
  • To reopen the pinned chat within ChatGPT when requested by you.

No data leaves your browser, and we do not perform any background tracking or analytics.

Permissions Explanation

The extension requests permission to run on https://chat.openai.com/* so it can add the "Pin" button to ChatGPT messages. This permission is used only for injecting the extension's interface on that site. The extension does not interact with any other websites or read unrelated data.

Your Choices

  • You can delete any saved pins from the extension at any time.
  • You can clear all extension data by using the "Clear Pins" option or by removing the extension.
  • You can manage permissions for PinGPT in Chrome's extension settings.

Data Security

All stored data is kept locally within your browser storage. PinGPT does not transfer data to remote servers, minimizing any external risk. However, if you share or export your pins manually, they are subject to the same privacy considerations as any personal data you handle.

📖 Detailed Usage Guide

Pin a Message (3 Easy Ways!)

Method 1: Fixed Pin Button (Easiest!)

  • Look for the "📌 Pin Message" button in the bottom-right corner of ChatGPT
  • Click it to pin the last assistant response
  • Perfect for quick pinning!

Method 2: Hover Method

  • Move your mouse over any ChatGPT message
  • A floating "📌 Pin" button should appear in the top-right of the message
  • Click it to pin that specific message

Method 3: Right-Click Method

  • Select any text in a ChatGPT message
  • Right-click → Choose "Pin selection to PinGPT"

🐛 Troubleshooting

  • If pin buttons don't appear, try refreshing the ChatGPT page
  • Make sure the extension is enabled in Chrome settings
  • Check that you're on chat.openai.com or chatgpt.com
  • If pins don't open correctly, the conversation URL might have changed
